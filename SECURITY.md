# Security Policy

## Reporting Security Issues

If you discover a security vulnerability, please email [your-email@example.com] directly. Do not open a public issue for security problems.

## Security Features

This application:
- Runs entirely on your local computer
- Does not send any data to external servers
- Does not collect user information
- Does not require an account or login

## Safe Usage

- Download only from the official GitHub Releases page
- Do not download from third-party websites
- Keep the application updated to the latest version
