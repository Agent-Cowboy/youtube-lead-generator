# Contributing

Thank you for your interest in this project.

## How to Contribute

Since this is proprietary software, code contributions are not accepted. However, you can help in other ways:

### Report Bugs
If you find a problem, open an issue with:
- What you were trying to do
- What happened instead
- Steps to reproduce the problem

### Suggest Features
Have an idea for improvement? Open an issue describing:
- The feature you want
- Why it would be useful

### Support the Project
- Star this repository
- Share the download link with others
- Consider donating to support development

## What You Cannot Do

- Submit code changes (pull requests will not be accepted)
- Fork and modify the software
- Redistribute the software

Thank you for understanding.
